🎵 MP3 Tag Reader & Editor

The MP3 Tag Reader & Editor is a command-line based C application designed to view and edit ID3v2 metadata tags in MP3 files.This project allows users to easily manage important audio file information such as Title, Artist, Album, Year, Genre, and Comments directly from the terminal.
The application uses file handling concepts in C to read and modify MP3 frame data efficiently while preserving the original audio content.

Features
🎵 View MP3 Tags – Display metadata information stored in MP3 files.

✏ Edit MP3 Tags – Modify tags like:

Title

Artist

Album

Year

Genre / Music Type

Comments

📂 MP3 Validation – Checks whether the given file is a valid MP3 file using the ID3 header.

💾 Safe File Editing – Uses a temporary file to prevent data corruption during updates.

⚡ Command-Line Interface – Lightweight and easy to use.

🧩 Endian Conversion Handling – Supports proper metadata frame size interpretation.

📖 Help Menu – Built-in usage guide for commands and options.
